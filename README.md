# Math630-codes

Codes for in-class collaboration for the course: Numerical Linear Algebra (Math 630) at the [University of Maryland Baltimore County](https://www.umbc.edu).

Codes are provided in Matlab, Python, and even some in C++.  To build the C++ codes, CMake version 3.10 or higher is required.  The Python codes require version 3, along with the Numpy and SciPy packages.
